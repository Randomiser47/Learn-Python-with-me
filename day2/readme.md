Master Python's fundamental data types and build practical calculators while learning essential programming concepts.

## 📚 Key Concepts Covered
- **Primitive Data Types**:  
  - `int`, `float`, `bool`, `str`
- **Type Conversion**:  
  - `str()` ↔ `int()` ↔ `float()`
- **Mathematical Operations**:  
  - PEMDAS order of operations  
  - Rounding with `round()`  
  - Floor division (`//`)
- **Formatted Strings (f-strings)**:  
  - Embedding variables in strings

## 🛠️ Projects Included
1. **BMI Calculator** (`BMI.py`)  
   Calculate Body Mass Index with proper type conversions.
2. **Life Time Calculator** (`daystill90.py`)  
   Convert age to days/weeks/months until 90.
3. **Tip Calculator** (`tip_calculator.py`)  
   Split bills with percentage-based tips.

## 🎯 Project Highlights

### 1. BMI Calculator
```python
height = 1.75  # meters
weight = 68.5  # kilograms
bmi = weight / height ** 2
# Output: 22
```

### 2. Life Time Calculator
```
Age: 25
Output: You have 23,725 days, 3,380 weeks, 780 months, 65 years left.
```

### 3. Tip Calculator
```
Total bill: $150
Tip percentage: 12%
People: 5
Output: Each person should pay: $33.60
```
